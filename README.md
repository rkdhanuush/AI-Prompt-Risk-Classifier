# AI-Prompt-Risk-Classifier
# 🛡️ AI Prompt Safety Analyzer

A lightweight web application that analyzes user prompts and classifies them into **High Risk**, **Medium Risk**, or **Safe** categories based on predefined AI safety keywords.

This project demonstrates basic AI safety concepts, prompt analysis, risk classification, and responsible AI usage using HTML, CSS, and JavaScript.

---

## 📌 Features

✅ Detects High Risk prompts

✅ Detects Medium Risk prompts

✅ Identifies Safe prompts

✅ Dynamic Risk Meter

✅ AI Safety Tips

✅ Prompt Risk Explanation

✅ Stores Last 5 Prompt History (Local Storage)

✅ Responsive User Interface

## ⚙️ How It Works

1. Enter a prompt into the text box.
2. Click **Analyze Prompt**.
3. The application checks the prompt against predefined keyword lists.
4. The prompt is classified as:

- 🔴 High Risk
- 🟡 Medium Risk
- 🟢 Safe

5. The application displays:

- Risk Level
- Risk Percentage
- AI Safety Tip
- Explanation
- Prompt History
